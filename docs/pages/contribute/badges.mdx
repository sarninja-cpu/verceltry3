# Contributor Badges

Our badge system recognizes and celebrates the contributions of our community members. Badges are awarded for achievements, milestones, and ongoing activity.

## Badge Categories

### Achievement Badges

These badges represent significant milestones and responsibilities. They are permanent and reflect historical contributions:

- **Framework-Steward** - Maintains and owns a specific framework
- **Core-Contributor** - Regular, trusted contributor with merge rights
- **Top-Reviewer** - Exceptional code review contributions
- **Contributor-5/10/25** - Milestone badges for contribution count
- **Reviewer-10/25** - Milestone badges for review count
- **Early-Contributor** - One of the first contributors to the project

### Activity Badges

These badges reflect recent activity and can change over time:

- **Active-Last-30d** - Active within the last 30 days
- **Active-Last-90d** - Active within the last 3 months
- **New-Joiner** - First contribution within the last 30 days
- **Dormant-90d+** - No activity for more than 90 days (optional)
